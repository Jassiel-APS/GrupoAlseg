Estructura del proyecto

assets/
  css/
    styles.css  # CSS principal
  js/
    script.js   # JS principal
  img/
    logo/       # logos originales
    illustrations/ # imágenes/ilustraciones
  fonts/

index.html

Instrucciones rápidas:
- Sirve `index.html` en un servidor local (Live Server o `python -m http.server`).
- Si editas `styles.css` o `script.js`, los archivos en `assets/css` y `assets/js` deben actualizarse.
