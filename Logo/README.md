Coloca aquí el logo oficial de la compañía (recomendado):

- Nombre de archivo recomendado: `logo.svg` (soporta redimensionado sin pérdida).
- Alternativa: `logo.png` o `logo@2x.png` para versiones raster.

Tamaños sugeridos:
- Header: 48×48 px (CSS controla el tamaño con `.nav-logo img`).
- Footer: 40×40 px.

Cómo reemplazar:
1. Copia tu archivo `logo.svg` dentro de esta carpeta `Logo/`.
2. Si usas `logo.png`, renómalo a `logo.png` y el HTML ya tiene fallback al `Logo/logo.svg` en caso de error.

Notas:
- Si subes un `logo.svg` con colores exactos, el sitio usará ese archivo automáticamente en header y footer.
- Si quieres que inserte el SVG inline para optimizar carga, pégame el contenido del SVG y lo inlineo en `index.html`.
